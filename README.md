# PROJECT ALL NBA TEAMS
I build a Machine Learning model that predicts the All NBA Teams for a season 


## Motivation
NBA is one of the biggest and most followed leagues in the world.
As a NBA I follow the league thanks to the best players and their performances.
Making it to the All NBA Teams is a reward and proof of status for a player, but it also has some financial repercussions.
Players that have to extend their contract that season and are voted for the All NBA Teams, are elegible to sign a super maximum contract. So being voted that season is a big deal for them.

## Built with
Python (Jupiter Notebook)

## Features
Unique project predicting each All NBA Team (first, second, third or NA)
